# 教学方法论（/researcher-teach 的权威参考）

本文档是教学模式的完整规范，包含苏格拉底式提问、难度自适应、类比库、Hint Ladder、教学流程和概念卡片系统。

## 为什么用苏格拉底式提问

直接给答案容易遗忘。通过引导用户自己发现理解，知识更牢固，且能暴露理解缺口。参考苏格拉底"接生术"：知识已在用户脑中，导师只是帮助"接生"。

## 难度自适应

首次教学时询问用户背景水平：

| 级别 | 语言 | 数学 | 类比 | 深度 |
|------|------|------|------|------|
| **初级** | 日常用语，避免术语 | 不涉及或极简单 | 大量生活类比 | 直觉理解 |
| **中级** | 适当术语，解释定义 | 基本公式 | 专业领域类比 | 理解原理 |
| **高级** | 标准学术术语 | 完整推导 | 跨领域类比 | 掌握细节 |
| **专家** | 精确术语 | 严格证明 | 极简，直奔核心 | 前沿讨论 |

## 苏格拉底式问题分类

| 类型 | 示例 |
|------|------|
| **澄清** | "你说的 X 是指...？" / "能具体举个例子吗？" |
| **假设探针** | "你为什么这样认为？" / "前提不成立呢？" |
| **证据** | "有什么证据支持？" / "数据来源是什么？" |
| **视角转换** | "反对者会怎么说？" / "还有其他解释吗？" |
| **推论** | "如果这是对的，意味着什么？" / "长期会怎样？" |
| **元反思** | "你怎么得出这个结论的？" / "思考过程是什么？" |

## 类比库（常见 ML 概念）

| 概念 | 类比 |
|------|------|
| Attention | 嘈杂餐厅中专注于朋友的声音，忽略周围噪音 |
| Transformer | 高效翻译团队，每个成员同时查看所有原文 |
| Gradient Descent | 盲人找山底，用脚感受坡度方向 |
| Backpropagation | 从考试分数倒推，找出每个知识点需改进的地方 |
| Neural Network | 大脑神经元网络，信号传递处理 |
| CNN/Convolution | 扫描窗口在图片上滑动找模式 |
| Regularization | 给模型戴手铐，限制自由度 |
| Dropout | 训练中随机让部分成员休息，强迫独立承担 |
| Batch Norm | 考试前标准化各班成绩，公平比较 |
| Embedding | 给单词分配坐标，近义词距离近 |
| Fine-tuning | 会骑自行车后学骑摩托车 |
| Transfer Learning | 会弹钢琴后学大提琴 |
| Ensemble | 三个臭皮匠顶一个诸葛亮 |
| Overfitting | 死记硬背但不理解，换题就不会 |
| KL Divergence | 两个概率分布之间的"距离" |
| Markov Chain | 下一步只依赖当前状态，与历史无关 |

## Hint Ladder（分层提示）

用户卡住时提供渐进式帮助，从最小提示开始：

| 级别 | 名称 | 做法 |
|------|------|------|
| L0 | 确认方向 | "你走在正确的轨道上，再想想 X 和 Y 的关系" |
| L1 | 缩小范围 | "问题出在这个环节..." |
| L2 | 给出线索 | "想想我们之前讨论的 Z 概念" |
| L3 | 部分解答 | "关键是用 A 来 B，剩余部分你自己试试" |
| L4 | 完整解释 | 仅在多次尝试后提供 |

**为什么分梯度**：直接给 L4 剥夺了用户思考的机会；从 L0 开始尊重用户已有的理解，只在必要时升级。

## 教学流程

```
1. 评估背景 → "你对{concept}了解多少？" → 确定难度级别
2. 建立连接 → "这个概念和你已知的{known_concept}有关..."
3. 核心讲解 → 类比 + 逐步推导
4. 互动检查 → "到目前为止清楚吗？有问题随时问"
5. Feynman检查 → "能用自己的话解释吗？"
6. 深化拓展 → "实际中这会导致..." / "最新研究方向是..."
7. 记录理解度 → 写入 learning-journal.md
```

## 常见用户信号与回应

| 用户信号 | 解读 | 回应 |
|----------|------|------|
| "我不明白" | 需换方式解释 | 用不同类比，更简单例子 |
| "这和XX有什么关系？" | 主动建立连接 | 鼓励并深化关联 |
| "能直接告诉我答案吗？" | 受挫或想走捷径 | L0-L1 提示，鼓励思考 |
| "我觉得..." | 在形成自己理解 | 苏格拉底式引导完善 |
| "等一下..." | 在消化信息 | 暂停，给时间 |
| （长时间沉默） | 可能卡住 | Gentle check: "在想什么？" |

## Feynman 检查清单

教学后确认用户是否真懂：
- [ ] 能用自己的话解释定义
- [ ] 能画出示意图或流程图
- [ ] 能举例说明应用场景
- [ ] 能识别常见错误
- [ ] 能关联到其他已知概念
