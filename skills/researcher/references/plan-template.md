# 研究计划模板

本文档是 Phase 1（制定研究计划）的模板和子问题分解启发式。

## 模板

```markdown
# 研究计划: {topic}
日期: {YYYY-MM-DD}

## 核心研究问题
{main_question}

## 子问题分解
1. {sub_question_1} — 基础概念与经典方法
2. {sub_question_2} — 最新进展与 SOTA
3. {sub_question_3} — 应用与局限性

## 搜索策略
- 学术数据库: Google Scholar, arXiv, Semantic Scholar
- 关键词: {keywords}
- 时间范围: {time_range}

## 范围决策
- 包含: ...
- 排除: ...
```

## 子问题分解启发式

- 时间维度：经典方法？最新进展？未来方向？
- 方法维度：有哪些技术路线？各路线的优缺点？
- 应用维度：主要应用场景？实际效果如何？
