# 引文网络分析（深度模式可选功能）

本文档是 `/researcher-deep` 中"引文网络分析"功能的参考。用户勾选此功能时加载。

## 功能说明

给定一篇种子论文，沿引文网络发现关联论文。类似 Connected Papers / ResearchRabbit。

## 三种扩展方向

| 方向 | 搜索策略 | 发现目标 |
|------|----------|----------|
| **Prior Works** | 搜索种子论文的参考文献 | 领域基础、经典工作 |
| **Derivative Works** | 搜索引用种子论文的后续研究 | 最新进展、改进方法 |
| **Similar Work** | 搜索共引/耦合论文 | 同主题的不同方法 |

## 执行流程

1. AskUserQuestion 确认种子论文（标题/DOI/URL）
2. 三方向各搜索 2-3 轮
3. 收集 → 去重 → 按相关度排序
4. 写入 `paper-outline.yaml`，进入逐篇阅读

## 引文态度标注

对关键发现标注引用态度：
- **支持** `[论文, 年份]` — 后续研究证实了该结论
- **矛盾** `[论文, 年份]` — 后续研究得出相反结果
- **提及** `[论文, 年份]` — 引用了该工作但未验证
